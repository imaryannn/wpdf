# wPDF – Web PDF Tool

A modern, browser-based tool for working with PDF files. No server required - all processing happens locally in your browser for maximum privacy and security.

## 🌐 Live Demo
**[https://imaryannn.github.io/wpdf/](https://imaryannn.github.io/wpdf/)**

## ✨ Features

### 📄 PDF Merger
- **Upload Multiple PDFs**: Select multiple PDF files at once
- **Drag & Drop Interface**: Intuitive file upload experience
- **Arrange Pages**: Reorder files before merging with drag-and-drop
- **Preview & Edit**: See all files before final merge
- **One-Click Download**: Get your merged PDF instantly

### ✂️ PDF Splitter
- **Split by Page Range**: Extract specific pages (e.g., 1-3, 5, 7-10)
- **Extract Each Page**: Split PDF into individual page files
- **Smart Packaging**: Automatically creates ZIP files for multiple pages
- **Flexible Input**: Support for complex page range expressions

### 🗜️ PDF Compressor
- **Multiple Compression Levels**: Low, Medium, High compression options
- **Quality Control**: Balance between file size and quality
- **Instant Processing**: Fast compression with visual feedback
- **Size Optimization**: Reduce PDF file sizes significantly

### 🖼️ Images to PDF
- **Multi-Format Support**: JPG, PNG, and other image formats
- **Visual Arrangement**: Drag and drop to reorder images
- **Page Size Options**: A4, Letter, Legal paper sizes
- **Orientation Control**: Portrait or Landscape layouts
- **Image Preview**: See exactly how your PDF will look

### 📸 PDF to Images
- **Page Selection**: Choose specific pages to extract
- **Visual Page Preview**: See thumbnails of all pages
- **Format Options**: Export as PNG or JPEG
- **Quality Control**: Adjustable image quality slider
- **Batch Selection**: Select all or individual pages

### 🔒 PDF Security
- **Password Protection**: Add real password encryption to PDFs
- **Password Removal**: Remove protection from existing PDFs
- **Strength Indicator**: Visual password strength feedback
- **Client-Side Processing**: Passwords never leave your device
- **Real Encryption**: Uses jsPDF for actual password protection

### 🎨 PDF Watermark
- **Text Watermarks**: Add custom text with full styling control
- **Image Watermarks**: Upload logos or images as watermarks
- **Perfect Preview**: Real-time canvas preview showing exact results
- **9-Point Positioning**: Precise watermark placement
- **Advanced Controls**: Opacity, rotation, scaling, and font options
- **Page Selection**: Apply to all pages, specific pages, or ranges
- **Professional Quality**: High-resolution output with perfect accuracy

### 📄 Docs to PDF
- **Multi-Format Support**: DOCX, PPTX, TXT, RTF, HTML conversion
- **Page Settings**: Customizable page size, orientation, and margins
- **Conversion Options**: Preserve formatting, headers, footers
- **Batch Processing**: Convert multiple documents at once
- **Quality Control**: Optimized output with font embedding options

## 🚀 Getting Started

### Online Usage
1. Visit [https://imaryannn.github.io/wpdf/](https://imaryannn.github.io/wpdf/)
2. Choose your desired PDF tool from the navigation
3. Upload your files using drag-and-drop or browse button
4. Configure settings as needed
5. Preview your changes (where available)
6. Process and download your files

### Local Development
```bash
# Clone the repository
git clone https://github.com/imaryannn/wpdf.git

# Navigate to project directory
cd wpdf

# Open in your preferred web server
# (or simply open index.html in a modern browser)
python -m http.server 8000
# or
npx serve .
```
